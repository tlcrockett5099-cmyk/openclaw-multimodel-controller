# Skills Library

> AI Skills and Gemini Gems in Openclaw MultiModel Controller v1.1.0

---

## What Are Skills?

**Skills** are pre-configured AI personas — system prompts that tell the AI to behave as a specialist. Activate a skill and every message in that conversation benefits from the AI's focused expertise.

Examples:
- **Code Reviewer** active → the AI audits your code for bugs, security flaws, and style violations
- **Budget Advisor** active → the AI frames all responses through financial planning best practices
- **Brainstormer** active → the AI generates wide, creative, divergent ideas on anything you ask

## Activating Skills

1. Open the **Skills** tab (✦ sparkles icon in navigation)
2. Browse by category using the horizontal filter tabs, or search by name/description
3. Click **Activate** on any free skill — it activates instantly
4. Click **🔒 Pro Only** on a locked skill — you'll be taken to Settings to unlock Pro
5. Active skills show a teal glow and "✓ Active" button

The number of active skills appears as a badge on the Skills icon in the sidebar and navigation.

---

## Skills in the Chat

When skills are active, a thin banner appears below the chat header:

```
🔧 Active Skills: Code Reviewer, Technical Writer
```

All active skill system prompts are injected at the start of every message you send — the AI automatically behaves as the specialist(s) you've activated.

Deactivate skills from the Skills page at any time.

## Creating Custom Skills

Custom skills are available to **all users** (free and Pro):

1. Scroll to the **Custom Skills** section at the bottom of the Skills page
2. Click **+ Add Custom**
3. Fill in:
   - **Name** — e.g., "Legal Drafter"
   - **Icon** — an emoji (e.g., ⚖️)
   - **Description** — one-sentence summary
   - **System Prompt** — the instruction sent to the AI
   - **Tags** — comma-separated keywords for search
4. Click **Save Skill**

Custom skills appear in the grid alongside built-in skills and can be activated or deleted at any time.
